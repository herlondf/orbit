# Changelog

All notable changes to Orbit are documented here.

## [1.0.0] — 2026-03-22

### Added
- 81 built-in services across 13 categories (Mensagens, Trabalho, Email, etc.)
- Brand SVG icons for 30+ services (WhatsApp, Slack, Gmail, etc.)
- Multiple isolated workspaces with per-workspace accent color
- Glassmorphism sidebar with gradient effects
- AES-256-GCM encryption for config files with master password
- AI Sidebar (ChatGPT, Claude, Gemini, Perplexity) — Ctrl+Shift+A
- Privacy Mode (blur overlay) — Ctrl+Shift+P
- Command palette — Ctrl+K
- Keyboard shortcuts cheatsheet — Ctrl+?
- Focus mode (sidebar toggle) — Ctrl+B
- Pop-out service windows
- Workspace scheduler (auto-switch by time/day)
- Reading list (save URLs for later)
- orbit:// URL scheme (deep links)
- Do Not Disturb with timer (15min/1h/4h/until tomorrow)
- Quiet Hours (scheduled DND with weekday selector)
- Notification history (last 50 entries)
- Per-service notification sounds
- Session time tracking with weekly stats dashboard
- Local backup (ZIP export/import)
- GitHub Gist cloud sync
- Rambox/Ferdium importer
- PIN lock screen with auto-lock timer
- Per-service incognito mode
- Per-service HTTP/SOCKS5 proxy
- Ad blocker (toggle in tray)
- Dark/Light/System themes with 6 accent colors
- Splash screen on launch
- System tray with badge count
- Google cookie bridge (bypass Google login block)
- Slack Electron compatibility layer (post-May 2026 readiness)
- MSIX + MSI packaging infrastructure
- Auto-updater with GitHub Releases integration
