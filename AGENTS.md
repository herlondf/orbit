# Guia Operacional - Repositorio em D:\IA

## Regra Principal
- Este repositorio usa `D:\IA\Skills` como repositorio padrao de skills.
- Antes de agir, verificar se existe uma skill aplicavel em `D:\IA\Skills`.
- Ao usar uma skill, seguir primeiro o `SKILL.md` correspondente.

## DevFlow
- Para fluxo de desenvolvimento, usar `D:\IA\Skills\devflow`.
- A fonte unica de verdade do DevFlow e `D:\IA\DevFlow.json`.

## Log obrigatorio
- Antes de qualquer acao de skill, emitir:
  - `[skill:<nome>] <acao curta>`
